# Kujang Core
Core of [generator-kujang]


# Kujang Generator
Kujang generator to build Flutter Apps using OpenAPI (Swagger 3.0) Standard

## Development

### Clone Kujang Core
```bash
git clone to/your/kujang-core
```
### Link Kujang Core
```bash
npm link to/your/kujang-core
```

### Unlink Kujang Core
```bash
npm unlink to/your/kujang-core
```
